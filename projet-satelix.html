Satelix Automation Suite v2.0

Suite d'automatisation d'inventaires et de déploiements

2024 - 2025 Projet Entreprise Production
Voir sur GitHub

Contexte et Objectifs

Contexte du projet

Dans le cadre de mon alternance, l'entreprise utilisait Satelix pour la gestion de son parc informatique et Sage 100 pour la gestion commerciale. La création manuelle des inventaires Satelix et leur synchronisation avec Sage 100 représentaient une tâche chronophage et répétitive, réalisée quotidiennement par les équipes techniques.

Face à ce constat, j'ai proposé de développer une suite d'automatisation complète permettant de générer automatiquement les inventaires, de les envoyer aux serveurs concernés, et de gérer les déploiements de configurations. Cette solution a permis de faire gagner plusieurs heures par jour à l'équipe IT.

Objectifs principaux

  • Automatiser la création d'inventaires Satelix pour éliminer les tâches manuelles répétitives
  • Planifier l'exécution quotidienne à heure fixe (8h00) sans intervention humaine
  • Garantir la compatibilité multi-OS (Windows et Linux) pour couvrir tout le parc serveurs
  • Offrir une interface utilisateur intuitive pour les techniciens moins à l'aise avec la ligne de commande
  • Intégrer des outils de diagnostic et réparation pour résoudre automatiquement les problèmes courants
  • Assurer la sécurité des identifiants via chiffrement et stockage local sécurisé

Bénéfices pour l'entreprise

  • Gain de temps : Économie de 2-3 heures par jour d'intervention manuelle
  • Fiabilité : Réduction des erreurs humaines dans la création des inventaires
  • Traçabilité : Logs détaillés de toutes les opérations effectuées
  • Simplicité : Aucune compétence technique requise pour utiliser l'outil
  • Portabilité : Fonctionnement depuis une clé USB sans installation

Automatisation Selenium - Connexion sécurisée

# Module de connexion automatique à Satelix
import os
from selenium import webdriver
from selenium.webdriver.common.by import By
from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC
from dotenv import load_dotenv
import logging

class SatelixConnector:
    def __init__(self):
        load_dotenv()  # Charge les credentials depuis .env
        self.driver = None
        self.logger = logging.getLogger(__name__)

    def connect(self):
        """Connexion automatique à l'interface Satelix"""
        try:
            # Configuration du driver Chrome en mode headless
            options = webdriver.ChromeOptions()
            options.add_argument('--headless')
            options.add_argument('--disable-gpu')
            self.driver = webdriver.Chrome(options=options)

            # Navigation vers Satelix
            self.driver.get(os.getenv('SATELIX_URL'))

            # Attente du chargement de la page de login
            wait = WebDriverWait(self.driver, 10)
            username_field = wait.until(
                EC.presence_of_element_located((By.ID, "username"))
            )

            # Saisie des identifiants (stockés en local chiffré)
            username_field.send_keys(os.getenv('SATELIX_USER'))
            password_field = self.driver.find_element(By.ID, "password")
            password_field.send_keys(os.getenv('SATELIX_PASS'))

            # Clic sur le bouton de connexion
            login_btn = self.driver.find_element(By.ID, "login-btn")
            login_btn.click()

            # Vérification de la connexion réussie
            wait.until(EC.url_contains("dashboard"))
            self.logger.info("Connexion à Satelix réussie")
            return True

        except Exception as e:
            self.logger.error(f"Erreur de connexion: {str(e)}")
            return False

    def create_inventory(self, inventory_type, data):
        """Création automatique d'un inventaire"""
        # Navigation vers le module d'inventaire
        self.driver.get(f"{os.getenv('SATELIX_URL')}/inventory/create")

        # Sélection du type d'inventaire
        type_select = self.driver.find_element(By.ID, "inventory-type")
        type_select.send_keys(inventory_type)

        # Remplissage des données (logique métier)
        for field, value in data.items():
            field_element = self.driver.find_element(By.NAME, field)
            field_element.send_keys(value)

        # Soumission du formulaire
        submit_btn = self.driver.find_element(By.ID, "submit-inventory")
        submit_btn.click()

        self.logger.info(f"Inventaire {inventory_type} créé avec succès")

Script de Planification Windows (Batch)

@echo off
REM Script de planification quotidienne Satelix
REM Exécution programmée à 8h00 via Windows Task Scheduler

echo [%date% %time%] Démarrage de l'automatisation Satelix
echo.

REM Activation de l'environnement virtuel Python
cd /d "%~dp0"
call venv\Scripts\activate

REM Exécution du script principal en mode silencieux
python main.py --mode silent --log-level INFO

REM Vérification du code de sortie
if %errorlevel% neq 0 (
    echo [ERREUR] L'automatisation a échoué (code: %errorlevel%)
    REM Envoi d'une notification par email
    python send_alert.py --error-code %errorlevel%
) else (
    echo [SUCCES] Automatisation terminée avec succès
)

REM Log de fin d'exécution
echo [%date% %time%] Fin de l'automatisation
echo.

pause
