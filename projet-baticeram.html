Contexte et Objectifs

Contexte du projet

Baticeram est une entreprise familiale spécialisée dans les chapes liquides, le chauffage au sol, l'isolation thermique et acoustique ainsi que la pose de carrelage et céramique. Fondée en 1995 et implantée en Vendée Nord et Maine-et-Loire Ouest, l'entreprise dispose de solides certifications (QUALIBAT, QB46, CQP Chef d'Équipe) mais ne disposait d'aucune présence en ligne.

Face à l'importance croissante du digital dans le secteur du BTP et à la recherche d'informations par les clients sur Internet, l'entreprise avait besoin d'un site vitrine professionnel pour :

  • Présenter ses services et son expertise technique
  • Valoriser ses réalisations et certifications
  • Faciliter la prise de contact pour les prospects
  • Améliorer sa visibilité locale (référencement naturel)

Objectifs principaux

  • Créer une vitrine web professionnelle reflétant le sérieux et l'expertise de l'entreprise
  • Développer un design responsive optimisé pour tous les appareils (mobile, tablette, desktop)
  • Mettre en avant les services proposés avec descriptions détaillées et visuels attractifs
  • Faciliter la prise de contact avec formulaires et coordonnées clairement affichés
  • Optimiser le référencement naturel (SEO) pour une meilleure visibilité locale
  • Garantir des performances optimales avec temps de chargement rapide et animations fluides

Contraintes du projet

  • Budget limité : Solution économique sans coûts d'hébergement récurrents
  • Simplicité de mise à jour : Structure permettant des modifications faciles
  • Compatibilité maximale : Support des navigateurs récents et anciens
  • Performances : Chargement rapide même avec connexions mobiles lentes

Animations au scroll avec Intersection Observer

// Animation au scroll des éléments
const observerOptions = {
    threshold: 0.1,
    rootMargin: '0px 0px -100px 0px'
};

const animateOnScroll = (entries, observer) => {
    entries.forEach(entry => {
        if (entry.isIntersecting) {
            entry.target.classList.add('animate-in');
            observer.unobserve(entry.target); // Animation une seule fois
        }
    });
};

const observer = new IntersectionObserver(animateOnScroll, observerOptions);

// Observer tous les éléments avec classe 'fade-in'
document.querySelectorAll('.fade-in').forEach(element => {
    observer.observe(element);
});

// Compteurs animés pour chiffres clés
const animateCounter = (element, target) => {
    let current = 0;
    const increment = target / 100;
    const timer = setInterval(() => {
        current += increment;
        if (current >= target) {
            element.textContent = target.toLocaleString('fr-FR');
            clearInterval(timer);
        } else {
            element.textContent = Math.floor(current).toLocaleString('fr-FR');
        }
    }, 20);
};

// Lancement au scroll des compteurs
document.querySelectorAll('.counter').forEach(counter => {
    const target = parseInt(counter.dataset.target);
    const counterObserver = new IntersectionObserver((entries) => {
        if (entries[0].isIntersecting) {
            animateCounter(counter, target);
            counterObserver.disconnect();
        }
    });
    counterObserver.observe(counter);
});

Menu burger responsive avec animations

// Gestion du menu mobile
const navToggle = document.querySelector('.nav-toggle');
const navMenu = document.querySelector('.nav-menu');
const body = document.body;

navToggle.addEventListener('click', () => {
    // Toggle classes pour animation
    navToggle.classList.toggle('active');
    navMenu.classList.toggle('active');
    body.classList.toggle('menu-open'); // Empêche scroll en arrière-plan

    // Animation burger en X
    const spans = navToggle.querySelectorAll('span');
    if (navToggle.classList.contains('active')) {
        spans[0].style.transform = 'rotate(45deg) translate(6px, 6px)';
        spans[1].style.opacity = '0';
        spans[2].style.transform = 'rotate(-45deg) translate(6px, -6px)';
    } else {
        spans[0].style.transform = 'none';
        spans[1].style.opacity = '1';
        spans[2].style.transform = 'none';
    }
});

// Fermeture du menu au clic sur un lien
document.querySelectorAll('.nav-link').forEach(link => {
    link.addEventListener('click', () => {
        navToggle.classList.remove('active');
        navMenu.classList.remove('active');
        body.classList.remove('menu-open');
    });
});

// Fermeture au scroll (mobile)
let lastScroll = 0;
window.addEventListener('scroll', () => {
    const currentScroll = window.pageYOffset;
    if (currentScroll > lastScroll && navMenu.classList.contains('active')) {
        navToggle.click(); // Ferme le menu en descendant
    }
    lastScroll = currentScroll;
});

// Système de lightbox pour galerie
class Lightbox {
    constructor() {
        this.lightboxOverlay = this.createLightboxElement();
        this.currentIndex = 0;
        this.images = [];
        this.initEventListeners();
    }

    createLightboxElement() {
        const overlay = document.createElement('div');
        overlay.className = 'lightbox-overlay';
        overlay.innerHTML = `
            
            
            
            
            

        `;
        document.body.appendChild(overlay);
        return overlay;
    }

    open(imageElement, galleryImages) {
        this.images = Array.from(galleryImages);
        this.currentIndex = this.images.indexOf(imageElement);
        this.show(this.currentIndex);
        this.lightboxOverlay.classList.add('active');
        document.body.style.overflow = 'hidden';
    }

    show(index) {
        const img = this.lightboxOverlay.querySelector('.lightbox-image');
        const caption = this.lightboxOverlay.querySelector('.lightbox-caption');
        img.src = this.images[index].src;
        img.alt = this.images[index].alt;
        caption.textContent = this.images[index].alt;
    }

    next() {
        this.currentIndex = (this.currentIndex + 1) % this.images.length;
        this.show(this.currentIndex);
    }

    prev() {
        this.currentIndex = (this.currentIndex - 1 + this.images.length) % this.images.length;
        this.show(this.currentIndex);
    }

    close() {
        this.lightboxOverlay.classList.remove('active');
        document.body.style.overflow = '';
    }

    initEventListeners() {
        this.lightboxOverlay.querySelector('.lightbox-close').addEventListener('click', () => this.close());
        this.lightboxOverlay.querySelector('.lightbox-next').addEventListener('click', () => this.next());
        this.lightboxOverlay.querySelector('.lightbox-prev').addEventListener('click', () => this.prev());
        this.lightboxOverlay.addEventListener('click', (e) => {
            if (e.target === this.lightboxOverlay) this.close();
        });

        // Navigation au clavier
        document.addEventListener('keydown', (e) => {
            if (!this.lightboxOverlay.classList.contains('active')) return;
            if (e.key === 'Escape') this.close();
            if (e.key === 'ArrowRight') this.next();
            if (e.key === 'ArrowLeft') this.prev();
        });
    }
}

// Initialisation
const lightbox = new Lightbox();
document.querySelectorAll('.gallery-image').forEach(img => {
    img.addEventListener('click', () => {
        lightbox.open(img, document.querySelectorAll('.gallery-image'));
    });
});

Conclusion et Apports

Le projet Baticeram illustre ma capacité à mener un projet web de bout en bout, de l'analyse des besoins au déploiement en production. Ce site vitrine professionnel a permis à l'entreprise d'avoir une présence en ligne moderne et efficace, sans coût d'hébergement récurrent.

Ce projet m'a particulièrement fait progresser sur :

  • Le développement frontend moderne avec HTML5, CSS3 et JavaScript vanilla
  • Le design responsive et l'approche mobile-first
  • L'optimisation des performances et du référencement naturel
  • La relation client et la compréhension des besoins métier
  • Le déploiement et l'hébergement de sites statiques

Le site est actuellement en ligne et continue d'évoluer selon les retours du client. Les bonnes pratiques de développement web appliquées garantissent sa maintenabilité et sa pérennité.
